# AccessPilot MVP - TASKS

> 目标：30 天内上线一个可收费的“电商网站无障碍合规扫描 + 报告生成”SaaS MVP。  
> 原则：每个任务只做一件事，完成后必须有明确产出。  
> 范围：只做 Web 版后台 + 扫描引擎 + 报告页 + 付费订阅，不做自动改站。

## 验收标准
- 用户能注册并添加一个站点。
- 用户能手动触发一次扫描。
- 系统能扫描 10 到 50 个页面并保存问题。
- 用户能看到健康分、问题列表和问题详情。
- 付费用户能导出 PDF 并接收邮件通知。
- 系统能稳定完成每周定时扫描。
